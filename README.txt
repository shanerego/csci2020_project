README

Completed by: Shane Rego (100623789) & Rolando Agullano (100622368)

Description: Purpose of this program was to mimic an email spam filter. Various 
tools were used including test/train files, which are included. 

Installation: Unzip folder into desired directory, then open using preferred text
editor (Atom was the original software to be used). To begin program, enter a 1 to
denote a test file is to be used, as well as a 0 to denote a train file once the program
compiles.

Known Bugs: If nothing is entered in command-line, IndexOutOfBound Error will occur.
